# Cardinal —— CTF AWD 线下赛平台

## 介绍

Cardinal 是由 Vidar-Team 开发的 AWD 比赛平台，使用 Go 编写。本程序可以作为 CTF 线下比赛平台，亦可用于团队内部 AWD 模拟练习。

## 特性

* 可控：针对线下的多种突发状况设置了预案

* 追溯：所有重要操作均有日志记录

* 准确：毫秒级计时，精准计算分数、排名
